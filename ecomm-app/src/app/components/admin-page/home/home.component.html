Admin Page

This section contain several parts to admin starting from :

User Managment

In this section you can manage :

  • Add new role to user
  • Remove existing role to user
  • Make user Active
  • Make user InActive

Main Menu

In this section you can manage main menu for the site by adding , updating and remove main menu

Sub Main Menu

In this section you can manage sub main menu for the site by adding , updating and remove sub main menu to main menu

Products

In this section you can manage products for the site by adding , updating and remove products to sub main menu

Items

In this section you can manage Items by doing the following actions

  • Add new item to the product
  • Update existing item to the product
  • Delete item from the product
  • Ammend the main pictures and additional pictures to the item
  • Ammend the properties of the items
